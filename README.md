게임방법

레벨업을 하면 더 많은 양의 물품을 다룰 수 있을겁니다.

개발자 모드를 사용하면

가격변동 원인을 제작할 수 있으며, 농작물 사진이 있다면 농작물을 더 추가할 수 있을겁니다.
